## 🤝 Metodología de Trabajo Colaborativo

Ambos autores hemos utilizado control de versiones (Git/GitHub) para estructurar este repositorio, separando los entornos de trabajo en directorios distintos para evitar conflictos de *merge*, y unificándolos a través de este índice principal mediante enlaces relativos en Markdown.
